## Description
A console-based Loan Banking Application developed using Core Java.
The application simulates real-world banking loan processing and
eligibility verification using Object-Oriented Programming principles.

## Loan Types Implemented
- Personal Loan
- Home Loan
- Car Loan
- Education Loan
- Gold Loan

## Features
- Aadhaar, PAN, and Mobile number validation using Regex
- Salary, age, and CIBIL score based eligibility checks
- Different interest rates for different loan types
- Loan-specific document verification
- Realistic banking workflow simulation

## OOP Concepts Used
- Inheritance
- Polymorphism (Method Overriding & Dynamic Method Dispatch)
- Encapsulation
- Abstraction

## Technologies Used
- Core Java
- OOP Concepts
- Regex
- Scanner Class
- Console I/O
